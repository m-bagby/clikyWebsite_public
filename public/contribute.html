Want to help make cliky better?

Cliky is open-source! Contributions are welcome from anyone who wants to help improve the program. Whether you're a developer, a designer, or just a fan, there are many ways you can contribute to the project.

Cliky's code is hosted on GitHub. Feel free to explore the code, submit pull requests, or open issues if you find any bugs or have ideas for new features.

Thank you for your interest in contributing to Cliky. Together, we can make this program even better for everyone.
